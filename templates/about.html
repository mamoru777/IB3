О программе

Работа выполнена Сенькиным А.Е. ПИбд-42

Группа: ПИбд-42

Вариант - 2 (Алгоритм DES в режиме CBC (Cipher Block Chaining) - цепочка блоков).

Алгоритм DES (Data Encryption Standard) в режиме CBC (Cipher Block Chaining) - это метод блочного шифрования, который обеспечивает дополнительный уровень безопасности за счет использования вектора инициализации и предварительного XOR-а блоков открытого текста.

Основные шаги алгоритма:

1. Инициализация:

- Выбирается секретный ключ K для DES.

- Генерируется случайный вектор инициализации (IV).

2. Разделение на блоки:

- Открытый текст разбивается на блоки фиксированного размера (обычно 64 бита).

3. Шифрование в режиме CBC:

- Используется IV для первого блока: CipherText[0] = DES(PlainText[0] XOR IV, K)

- Для каждого следующего блока: CipherText[i] = DES(PlainText[i] XOR CipherText[i-1], K)

- Получается поток зашифрованных данных.

5. Передача или хранение данных:

- Зашифрованный текст может быть передан или сохранен.

Расшифровка в режиме CBC:

- Используется тот же ключ K и тот же IV для инициализации.

- Для каждого блока: PlainText[i] = DES^-1(CipherText[i], K) XOR CipherText[i-1]

- Получается исходный открытый текст.

Режим CBC обеспечивает криптографическую стойкость за счет комбинации ключа, вектора инициализации и предыдущего зашифрованного блока для каждого блока открытого текста, что делает его устойчивым к некоторым типам атак.

Возможности для сохранения и выбора обрабатываемого файла:

- Пользователь выбирает и загружает файл из системы на сервер либо пишет и загружает на сервер текст

- После обработки файла или текста на сервере, то есть после шифрования, текст или файл сохраняется на сервере и пользователь автоматически перенаправляется для скачивания обработанного файла или текста
